Objetivos del Ciclo

El ciclo de Desarrollo de Aplicaciones Web tiene como objetivos:

  1. Desarrollar habilidades en la creación de aplicaciones que funcionen en múltiples plataformas.
  2. Comprender los principios fundamentales de la programación móvil y web.
  3. Integrar tecnologías actuales para la creación de aplicaciones versátiles.

Contenidos del Ciclo

El ciclo se compone de varios módulos que cubren diferentes aspectos del desarrollo Web:

  • Módulo 1: Introducción al Desarrollo Web
      • Este módulo proporciona los conceptos básicos y fundamentales del desarrollo Web.
  • Módulo 2: Programación Web
      • En este módulo, los estudiantes aprenderán sobre las tecnologías web esenciales para el desarrollo.
  • Módulo 3: Desarrollo de paginas web
      • Se explorarán las herramientas y técnicas necesarias para desarrollar paginas web efectivas.
  • Módulo 4: Bases de Datos para Aplicaciones Web
      • Este módulo abordará la gestión de bases de datos para aplicaciones que funcionan en diferentes plataformas.
  • Módulo 5: Diseño de Interfaz de Usuario
      • Los estudiantes aprenderán principios de diseño centrados en el usuario para crear interfaces intuitivas y atractivas.
